Answer: List

Question 4: What is the result of `5 // 2`?

Answer: 2

Explanation: `//` is floor division; it returns the largest whole number less than or equal to the result.

Question 5: Which of the following is a mutable data type?

Answer: list

Explanation: Lists are mutable in Python, allowing modifications.

Question 6: How do you insert a comment in Python?

Answer: # comment

Explanation: Python uses the `#` symbol to indicate a comment.

Question 7: Which of the following is used to handle exceptions in Python?

Answer: try-except

Explanation: Python uses `try-except` blocks to handle exceptions.

Question 8: What keyword is used to define a function in Python?

Answer: def

Explanation: The `def` keyword is used to define functions in Python.

Question 9: What does `range(3)` return?

Answer: [0, 1, 2]

Explanation: `range(3)` generates numbers starting from 0 up to (but not including) 3.
