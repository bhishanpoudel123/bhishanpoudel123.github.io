    Qn 1: Which function in Pandas is used to detect missing values?

    Answer

    isnull()

    Explanation

    The isnull() function is used to detect missing values in a DataFrame.

    Qn 2: What does the `dropna()` function do?

    Answer

    Drops rows with missing values

    Explanation

    dropna() removes rows (or columns) that contain missing values.

    Qn 3: Which method is used to fill missing values with the mean of the column?

    Answer

    fillna(mean)

    Explanation

    fillna() is used to fill missing values, and you can pass the column mean to it.
