    Qn 1: What is the purpose of the `WITHOUT ROWID` clause in SQLite?

    Answer

    To create a table without the implicit ROWID column

    Explanation

    WITHOUT ROWID creates a table without the implicit ROWID, useful for certain optimizations.

    Qn 2: Which function would you use in SQLite to get the current timestamp?

    Answer

    CURRENT_TIMESTAMP

    Explanation

    CURRENT_TIMESTAMP returns the current date and time in SQLite.

    Qn 3: What is the default data type of a column in SQLite if not specified?

    Answer

    NONE

    Explanation

    If no type is specified, SQLite assigns it an affinity of NONE.

    Qn 4: How are boolean values stored in SQLite?

    Answer

    As 1 and 0 integers

    Explanation

    SQLite does not have a separate BOOLEAN type; it uses integers 1 (true) and 0 (false).

    Qn 5: Which of the following is true about SQLite's `VACUUM` command?

    Answer

    It compacts the database file

    Explanation

    VACUUM rebuilds the database file to defragment it and reduce its size.

    Qn 6: Which SQLite command lists all tables in the database?

    Answer

    SELECT * FROM sqlite_master WHERE type='table'

    Explanation

    SQLite uses the sqlite_master table to store metadata about the database, including table names.

    Qn 7: Which SQLite command allows you to see the schema of a table?

    Answer

    .schema

    Explanation

    .schema is a command in the SQLite shell that shows the schema for tables.

    Qn 8: How does SQLite handle foreign key constraints by default?

    Answer

    They are off by default and must be enabled

    Explanation

    SQLite supports foreign keys, but enforcement must be enabled with PRAGMA foreign_keys = ON.

    Qn 9: How does SQLite implement AUTOINCREMENT?

    Answer

    Using INTEGER PRIMARY KEY

    Explanation

    SQLite uses INTEGER PRIMARY KEY AUTOINCREMENT to create an auto-incrementing ID.

    Qn 10: What pragma statement turns on write-ahead logging in SQLite?

    Answer

    PRAGMA journal_mode = WAL

    Explanation

    PRAGMA journal_mode = WAL enables write-ahead logging in SQLite.
